# c语言编译器
学习的url https://lotabout.me/2015/write-a-C-interpreter-3/
## 设计
编译器三个步骤：
1. 词法分析，字符串转换为 结构
2. 语法分析，将词法分析得到的标记流生成一颗语法树
3. 目标代码生成，将语法树转换成目标代码
   
我们要做的三点：
1. 构建虚拟机和指令集，虚拟机来运行这个指令集
2. 构建词法分析器
3. 构建语法分析器
   
大致框架，四个函数：
1. ```next()``` 词法分析
2. ```program()```语法分析
3. ```expression(level)``` 解析表达式
4. ```eval()``` 虚拟机入口，解释目标代码。
